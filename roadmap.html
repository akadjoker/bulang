v1.0 - Foundation
Q1 2025
The initial release of BuLang with core VM functionality and essential features for game scripting.
  • Stack-based bytecode VM
  • Basic data types and operators
  • Functions and closures
  • Class-based OOP
  • Process system (50K+ concurrent)
  • Reference counting memory management
  • Raylib integration
  • API documentation
In Progress
v1.1 - Polish & Tools
Q2 2025
Performance improvements, debugging tools, and ecosystem expansion with better developer experience.
  • Debugger with breakpoints
  • Profiler for performance analysis
  • REPL interpreter
  • Package manager (BuPkg)
  • Visual Studio Code extension
  • JIT compilation support
  • Error messages improvements
  • Comprehensive error handling
Planned
v2.0 - Advanced Features
Q3-Q4 2025
Major feature expansion including async/await, improved type system, and WASM support.
  • Async/await patterns
  • Optional type annotations
  • Generic types support
  • Trait system
  • WASM compilation target
  • FFI improvements
  • Module system enhancements
  • Standard library expansion
Planned
v3.0 - Ecosystem Maturity
2026+
Focus on ecosystem maturity with official frameworks, IDE support, and enterprise features.
  • Official game framework
  • Physics engine (2D/3D)
  • Advanced graphics API
  • Network programming support
  • Database integration
  • Multiplayer support
  • Cloud integration
  • Advanced profiling tools
