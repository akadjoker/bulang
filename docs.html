Introduction to BuLang

BuLang is a stack-based bytecode virtual machine designed specifically for game development. It provides high-performance script execution with support for object-oriented programming, first-class functions, and a unique process system for cooperative multitasking.

The language combines simplicity with power, allowing you to write game logic, AI behaviors, and interactive systems that run efficiently even with thousands of concurrent processes.

Getting Started

BuLang is designed to be easy to learn for developers coming from any programming background.

Hello World

Start with the simplest program:

// Hello World print("Hello, BuLang!")

The print() function outputs text to the console. That's all you need to get started!

Variables and Basic Types

BuLang uses dynamic typing - you don't need to declare variable types:

// Variables name = "BuLang" version = 1.0 count = 42 is_active = true print(name) print(count)

Language Basics

Data Types

BuLang supports the following primitive types:

Type Example Description
Number 42, 3.14 Integer or floating-point numbers
String "hello" Text values
Boolean true, false Truth values
Null nil Absence of value
Array [1, 2, 3] Collections of values
Object {x: 10, y: 20} Key-value pairs

Operators

// Arithmetic a = 10 + 5 // 15 b = 10 - 5 // 5 c = 10 * 5 // 50 d = 10 / 5 // 2 e = 10 % 3 // 1 // Comparison a == b // false a != b // true a > b // true a >= b // true // Logical true and false // false true or false // true not true // false

Control Flow

// If statements x = 10 if x > 5 { print("x is greater than 5") } elif x == 5 { print("x is 5") } else { print("x is less than 5") } // Loops for i = 0; i < 5; i = i + 1 { print(i) } // While loop count = 0 while count < 3 { print(count) count = count + 1 }

Functions

Functions in BuLang are first-class citizens. You can define them and pass them as arguments:

// Function definition def add(a, b) { return a + b } result = add(5, 3) print(result) // 8 print(apply(add, 10, 20)) // 30

Classes & Object-Oriented Programming

BuLang supports class-based OOP with methods and inheritance:

// Class definition class Player { def new(name, health) { this.name = name this.health = health } def take_damage(amount) { this.health = this.health - amount } def heal(amount) { this.health = this.health + amount } def status() { print(this.name + " - HP: " + this.health) } } // Using classes player = Player.new("Hero", 100) player.take_damage(20) player.status() // Hero - HP: 80

Processes - Cooperative Multitasking

BuLang's unique process system allows you to handle thousands of concurrent tasks efficiently:

// Process example process enemy_behavior(enemy_id) { while true { // Do something print("Enemy " + enemy_id + " acting") // Yield to next frame yield // Decision making if some_condition { break } } } // Spawn processes enemy_behavior(1) enemy_behavior(2) enemy_behavior(3)
Note: Processes use yield to pause execution and resume in the next update cycle. This enables smooth game loop integration.

Raylib Integration

BuLang integrates seamlessly with Raylib for 2D/3D graphics:

// Simple Raylib game loop def main() { InitWindow(800, 600, "BuLang Game") SetTargetFPS(60) while not WindowShouldClose() { BeginDrawing() ClearBackground({r: 0, g: 0, b: 0}) // Draw rectangle DrawRectangle(100, 100, 50, 50, {r: 255, g: 0, b: 0}) EndDrawing() } CloseWindow() } main()

Core Functions
